# PhpArray

Клас‑завантажувач перекладів з PHP‑масивів (`.php`‑файли, що повертають масив рядків). Наслідує `TranslationLoaderAbstract` і реалізує метод `load`, який забезпечує підключення файлу та валідацію вмісту.

## Методи

| Підпис                                | Повертаємий тип | Доступ | Призначення                                                                                                                                                                                        |
| ------------------------------------- | --------------- | ------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `load(string $filename): ArrayObject` | `ArrayObject`   | public | Перевіряє файл через `validateFile`, підключає його через замикання `require`, переконується, що результат — масив, інакше кидає `ETypeException`, та повертає `ArrayObject`. |

## Приклад файлу перекладів `ru.php`

```php
<?php

return [
    'greeting' => 'Привіт',
    'farewell' => 'Бувай',
];
```

## Приклад використання

```php
<?php

use Scaleum\i18n\Loaders\PhpArray;

$loader = new PhpArray(); // змінна lowerCamelCase

$messages = $loader->load(__DIR__ . '/messages/ru.php');

echo $messages['greeting']; // Привіт
```

## Практичні рекомендації

* Файл **повинен повертати** масив, а не виводити його — використовуйте `return [...];`.
* Ключі чутливі до регістру; дотримуйтеся єдиного стилю (`snake_case` або `dot.notation`).
* При великій кількості звернень обгорніть завантажувач декоратором‑кешем, щоб не підключати файл на кожен запит.
* Винятки `EInOutException` та `ETypeException` допоможуть швидко виявити проблеми з доступом до файлу або неправильним форматом даних.
