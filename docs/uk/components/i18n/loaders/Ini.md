# Ini

Клас‑завантажувач перекладів із `.ini`‑файлів. Наслідує `TranslationLoaderAbstract` і використовує `parse_ini_file` для перетворення даних у `ArrayObject`.

## Методи

| Підпис                                             | Повертаємий тип | Доступ    | Призначення                                                                                              |
| -------------------------------------------------- | --------------- | --------- | ------------------------------------------------------------------------------------------------------- |
| `load(string $filename): ArrayObject`              | `ArrayObject`   | public    | Перевіряє файл через `validateFile`, зчитує вміст `parse_ini_file`, перетворює у `ArrayObject`.          |
| `flatten(array $data, string $prefix = ''): array` | `array`         | protected | Рекурсивно перетворює багаторівневий масив у плоский («section.key»). Використовується всередині `load`. |

## Приклад використання

```php
<?php

use Scaleum\i18n\Loaders\Ini;

$loader = new Ini(); // змінна lowerCamelCase

$messages = $loader->load(__DIR__ . '/messages/ru.ini');

echo $messages['greeting']; // Привіт
```

## Приклад `ru.ini`

```ini
[greetings]
hello = "Привіт"

[farewell]
bye = "Бувай"
```

## Практичні рекомендації

* **Секції** в `.ini`‑файлі автоматично об’єднуються з ключами через крапку (`section.key`).
* **Типізація**: `INI_SCANNER_TYPED` зберігає числові та логічні значення. Для рядкових повідомлень це безпечно, але враховуйте при валідації.
* **Кеш**: як і інші завантажувачі, рекомендується обгортати декоратором‑кешем при великій кількості звернень.
