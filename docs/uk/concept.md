# Концепція використання Scaleum Framework

Scaleum — це модульний PHP-фреймворк, створений для побудови масштабованих, розширюваних і строго організованих застосунків. Він не нав’язує архітектуру, але надає чіткі механізми для її реалізації.

## Основні принципи

- **Контейнеризація логіки** — кожна функціональність реалізується у вигляді модуля (`Module`), ізольованого за простором імен і відповідальністю.
- **Чиста архітектура** — розділення шарів: моделі, сервіси, контролери, подання.
- **Конфігурація через код** — перевага конфігуратору, а не YAML/ENV/анотаціям.
- **Тестованість за замовчуванням** — всі залежності впроваджуються явно, можна легко мокати.
- **Мінімальна залежність від фреймворку** — більшість коду переносима.

Детальніше про архітектуру дивіться [тут](./architecture.md).
