# Вступ

**Scaleum Framework** — це модульний PHP-фреймворк, створений для побудови масштабованих, розширюваних та строго організованих застосунків. Він не нав’язує архітектуру, але надає чіткі механізми для її реалізації.

Основні принципи:
- Проста завантаження та конфігурація
- Модульна структура (`src/`)
- Вбудований DI, події, middleware, сесії, логування
- Підтримка CLI та HTTP-контексту

Докладніше про архітектуру дивіться [тут](./architecture.md).
