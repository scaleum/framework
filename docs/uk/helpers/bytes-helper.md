# BytesHelper

`BytesHelper` — утилітарний клас для роботи з розмірами даних у байтах, кілобайтах та інших одиницях.

## Призначення

- Переведення байтів у зручний для читання формат
- Конвертація між різними одиницями зберігання
- Витяг числового значення зі рядка з розміром
- Форматування розміру у рядок

## Основні методи

| Метод | Призначення |
|:------|:-----------|
| `bytesAssoc(int $size): mixed` | Перевести байти у масив `[розмір, одиниця виміру]` |
| `bytesStr(int $size, string $format = '%d%s'): mixed` | Перевести байти у форматований рядок |
| `bytesNumber(string $str): int\|float` | Витягти число байтів із рядка з одиницею виміру |
| `bytesTo(int $size, string $from = 'b', string $to = 'kb'): float` | Перевести між одиницями виміру |

## Підтримувані одиниці

- `b` — байти
- `kb` — кілобайти
- `mb` — мегабайти
- `gb` — гігабайти
- `tb` — терабайти
- `pb` — петабайти
- `eb` — ексабайти
- `zb` — зетабайти
- `yb` — йотабайти

---

## Приклади використання

### Перевести байти у рядок

```php
echo BytesHelper::bytesStr(1048576); // "1mb"
```

### Отримати масив: значення + одиниця
```php
$result = BytesHelper::bytesAssoc(1536);
// [1.5, 'kb']
```

### Конвертувати рядок розміру у байти
```php
$bytes = BytesHelper::bytesNumber('10MB');
// 10485760
```

### Перевести між одиницями
```php
$sizeInGb = BytesHelper::bytesTo(1048576, 'kb', 'gb');
// 1
```

## Особливості
- Розумна конвертація рядків без суворої вимоги до регістру (`10Mb`, `10mb`, `10MB` — все коректно).
- При передачі некоректної одиниці в `bytesNumber()` повертається 0.
- Внутрішнє округлення значень до трьох знаків після коми (`round(..., 3)`).
