# Registry

**Registry** — універсальний клас для зберігання та доступу до вкладених даних з використанням ключів та роздільників.

## Призначення

- Управління структурованими даними в пам’яті
- Підтримка вкладених ключів через роздільник
- Гнучке встановлення, отримання, видалення та об’єднання даних

## Основні можливості

| Метод | Призначення |
|:------|:-----------|
| `__construct(array $items = [], string $separator = '/')` | Ініціалізація реєстру даними та встановленням роздільника |
| `get(string $str, mixed $default = null)` | Отримання значення за ключем |
| `set(string $str, mixed $value)` | Встановлення значення за ключем |
| `has(string $str)` | Перевірка наявності значення за ключем |
| `merge(array $items, ?string $key = null)` | Об’єднання масиву з поточними даними |
| `unset(string $str)` | Видалення значення за ключем |
| `getItems()` | Отримання всіх значень реєстру |
| `setItems(array $items)` | Встановлення значень реєстру |
| `getSeparator()` | Отримання поточного роздільника |
| `setSeparator(string $separator)` | Встановлення нового роздільника |

## Приклади використання

### Створення реєстру та встановлення значень

```php
$registry = new Registry();
$registry->set('database/host', 'localhost');
$registry->set('database/port', 3306);
```

### Отримання значення

```php
$host = $registry->get('database/host');
```

### Перевірка існування ключа

```php
if ($registry->has('database/port')) {
    echo 'Порт бази даних задано.';
}
```

### Видалення значення

```php
$registry->unset('database/host');
```

### Об’єднання масивів

```php
$registry->merge([
    'cache' => ['enabled' => true, 'driver' => 'redis']
]);
```

### Робота з іншим роздільником

```php
$registry->setSeparator('.');
$registry->set('app.config.debug', true);
```

## Особливості

- Підтримка вкладених ключів з динамічним створенням проміжних вузлів.
- Безпечне об’єднання масивів за допомогою `ArrayHelper::merge`.
- Гнучка робота з конфігураціями.

## Винятки

- Немає специфічних винятків у базовій реалізації.
