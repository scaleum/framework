# Collection

**Collection** — універсальний клас для роботи з колекціями даних у стилі масивів та об’єктів.

## Призначення

- Управління наборами даних (елементами)
- Підтримка інтерфейсів `Iterator`, `ArrayAccess`, `Countable`
- Серіалізація у рядок або XML
- Гнучкі методи доступу та навігації

## Основні можливості

| Метод | Призначення |
|:------|:-----------|
| `__construct(?array $array = null)` | Ініціалізація колекції даними |
| `__toArray()` | Перетворення колекції у масив |
| `__toString()` | Серіалізація колекції у рядок |
| `__toXml()` | Перетворення колекції у XML |
| `set($key, $item)` | Встановлення елемента |
| `get($key, $default = null)` | Отримання елемента за ключем |
| `remove($key)` | Видалення елемента |
| `clear()` | Очищення всіх елементів |
| `count()` | Кількість елементів |
| `exists($key)` | Перевірка існування ключа |
| `isEmpty()` | Перевірка порожнечі колекції |
| `asort()`, `ksort()`, `sort()` | Сортування колекції |
| `merge(array $array, bool $overwrite = false)` | Об’єднання колекції з масивом |
| `fetch()` | Отримання поточного елемента та зсув вказівника |
| `seek($key)` | Пошук елемента за ключем |
| `uasort($callback)` | Сортування за допомогою користувацької функції |

## Приклади використання

### Створення колекції

```php
$collection = new Collection(['one' => 1, 'two' => 2]);
```

### Додавання елемента

```php
$collection->set('three', 3);
```

### Отримання елемента

```php
$value = $collection->get('one'); // 1
```

### Перевірка існування ключа

```php
if ($collection->exists('two')) {
    echo 'Є ключ two';
}
```

### Перетворення у масив

```php
$array = $collection->__toArray();
```

### Перетворення у XML

```php
$xml = $collection->__toXml();
echo $xml;
```

### Очищення колекції

```php
$collection->clear();
```

### Ітерація по колекції

```php
foreach ($collection as $key => $value) {
    echo "$key: $value\n";
}
```
