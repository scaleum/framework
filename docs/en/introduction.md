#  Introduction

**Scaleum Framework** is a modular PHP framework designed for building scalable, extensible, and strictly organized applications. It does not impose an architecture but provides clear mechanisms for its implementation.

Key principles:
- Simple loading and configuration
- Modular structure (`src/`)
- Built-in DI, events, middleware, sessions, logging
- Support for CLI and HTTP context

For more details on the architecture, see [here](./architecture.md).
