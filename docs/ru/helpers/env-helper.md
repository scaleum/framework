# EnvHelper

`EnvHelper` — утилитарный класс для безопасной работы с переменными окружения (`environment variables`).

---

## Назначение

- Получение переменных окружения с дефолтным значением
- Установка переменных окружения во время выполнения скрипта

---

## Основные методы

| Метод | Назначение |
|:------|:-----------|
| `has(string $key): bool` | Проверить, существует ли переменная окружения |
| `get(string $key, mixed $default = null): mixed` | Получить значение переменной окружения |
| `set(string $key, mixed $value): void` | Установить значение переменной окружения |
| `interpolateArray(array $items, array $options = []): array` | Рекурсивно интерполировать плейсхолдеры в массиве |
| `interpolateString(string $value, array $options = []): string` | Интерполировать плейсхолдеры в строке |

---

## Примеры использования

### Получить значение переменной

```php
$databaseHost = EnvHelper::get('DB_HOST', 'localhost');
```
Если переменная окружения `DB_HOST` отсутствует, будет возвращено 'localhost'.

### Установить значение переменной
```php
EnvHelper::set('APP_DEBUG', true);
```

### Проверить наличие переменной
```php
if (EnvHelper::has('APP_DEBUG')) {
	// переменная существует, даже если значение пустая строка
}
```

### Интерполировать плейсхолдеры в конфиге
```php
$config = [
	'db' => [
		'host' => '${DB_HOST}',
		'port' => '${DB_PORT:-5432}',
	],
];

$resolved = EnvHelper::interpolateArray($config, [
	'variables' => ['DB_HOST' => 'localhost'],
]);
```

### Получить значение переменной
```php
$isDebug = EnvHelper::get('APP_DEBUG', false);
// true
```

## Особенности
- Использует функции `getenv()` и `putenv()`
- Также использует `$_ENV` для согласованного доступа во время выполнения
- Значения автоматически конвертируются в строку при установке
- Сохраняет значения `''` и `'0'` (не подменяет их на дефолт)
- Возвращает дефолт только если переменная действительно отсутствует
- Поддерживает синтаксис плейсхолдеров: `${VAR}`, `${VAR:-default}`, `${VAR:?message}`
- Поддерживает простую работу без зависимости от дополнительных библиотек
